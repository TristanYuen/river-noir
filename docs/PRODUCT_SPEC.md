# River Noir／暗河德州产品与技术规范

## 1. 产品目标

River Noir 是一款面向朋友聚会的私人房德州扑克网页游戏。首个可交付版本在浏览器中支持完整本地人机对局，并提供稳定的联机通信边界。服务器接入后可用于 2～10 人朋友对局与 AI 混合入座。

## 2. 核心范围

- 本地训练：1 名真人与 2～9 名 AI。
- 联机房间：2～10 名真人，允许 AI 补位。
- 完整 No-Limit Texas Hold'em 流程。
- Fold、Check、Call、Bet、Raise、All-in。
- 主池、多层边池、平分底池和零散筹码。
- 三档 AI、胜率分析、牌力说明与公开信息范围提示。
- 简体中文与英文即时切换。
- 桌面优先，兼容手机横屏与竖屏。
- 虚拟筹码，无任何真实货币功能。

## 3. MVP 房间默认值

- 6 个座位。
- 初始筹码：10,000。
- 小盲／大盲：50／100。
- 行动时间：30 秒。
- AI 难度：标准。
- 本地训练默认开启胜率分析。
- 朋友房默认关闭实时胜率分析。

## 4. 视觉规范

- 背景：近黑色，带克制的径向暖光。
- 牌桌：深墨绿绒面，深色木质或金属包边。
- 强调色：香槟金，避免大面积高饱和金色。
- 文本：暖白主文字，灰金次级文字。
- 红色只用于红桃、方块和高风险提示。
- 动效节奏克制，使用淡入、位移、筹码聚合和庄家标记移动。
- 字体优先使用系统字体，品牌标题使用高对比衬线字体回退栈。

## 5. 架构边界

```text
React UI
  ↓
Player View Store
  ↓
GameTransport
  ├─ LocalTransport → Local Game Orchestrator → Shared Engine
  └─ WebSocketTransport → Remote Server
```

共享规则引擎只接收命令并返回新状态与领域事件。服务端可以直接复用，也可以根据协议和测试向量在其他语言中实现。

## 6. 完成标准

- 规则引擎覆盖完整牌局和全部已列边界规则。
- 本地模式能够连续完成多手牌。
- 3～10 人 AI 对局无死锁、无重复牌、筹码守恒。
- 双语界面不存在硬编码漏翻文本。
- 其他玩家底牌不进入当前玩家公开快照。
- 联机协议、错误码和重连策略文档完整。
- `npm run verify` 通过。
- 生产构建可部署至 GitHub Pages、Cloudflare Pages、Vercel 或任意静态服务器。
